# Generador de Códigos Únicos

Este repositorio contiene una página web que genera un código único con caracteres alfanuméricos. El código se genera al presionar un botón en la página.

## ¿Cómo usarlo?

1. Clona o descarga este repositorio.
2. Sube los archivos a GitHub y habilita GitHub Pages desde las configuraciones del repositorio.
3. Visita la URL proporcionada por GitHub Pages para acceder a la página y generar códigos.

## Cómo generar el código QR:

1. Obtén la URL de tu página desde GitHub Pages.
2. Usa un generador de códigos QR, como [QR Code Generator](https://www.qr-code-generator.com/), para crear el código QR que apunte a esa URL.
